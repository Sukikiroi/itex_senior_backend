## Description

an API for a Doctor's Office Management System. The system should
allow doctors and administrative staff to manage appointments, patient records, prescriptions, and
medical histories.


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Docker Env
```bash
$ docker-compose build
$ docker-compose up

```
